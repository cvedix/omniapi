# YOLOv11 Detection Solution (ONNX)

Solution phát hiện đối tượng sử dụng YOLOv11 với model ONNX.

## Ví dụ

- `example_yolov11_detection.json` - YOLOv11 object detection với ONNX model

## Tham số

- `MODEL_PATH`: Đường dẫn đến YOLOv11 ONNX model (`.onnx`)
- `SAVE_DIR`: Thư mục lưu output (optional)
